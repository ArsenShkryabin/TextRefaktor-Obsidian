# Руководство по вкладу в проект

Спасибо за интерес к улучшению Text Enhancer! 

## Как внести вклад

### Сообщения об ошибках

1. Проверьте, не создан ли уже issue с такой же проблемой
2. Создайте новый issue с описанием:
   - Что произошло
   - Шаги для воспроизведения
   - Ожидаемое поведение
   - Версия Obsidian и плагина

### Предложения

1. Создайте issue с меткой "enhancement"
2. Опишите вашу идею подробно
3. Объясните, как это улучшит плагин

### Pull Requests

1. Форкните репозиторий
2. Создайте ветку для вашей функции (`git checkout -b feature/amazing-feature`)
3. Внесите изменения
4. Убедитесь, что код компилируется (`npm run build`)
5. Создайте Pull Request

## Стандарты кода

- Используйте TypeScript
- Следуйте существующему стилю кода
- Добавляйте комментарии для сложной логики
- Обновляйте документацию при необходимости

## Тестирование

Перед отправкой PR убедитесь:
- [ ] Код компилируется без ошибок
- [ ] Все функции работают
- [ ] Тестовый режим работает
- [ ] Нет ошибок в консоли

## Вопросы?

Создайте issue с вопросом, и мы поможем!
