# Установка плагина на Windows

## Быстрая установка

1. **Соберите плагин:**
   ```powershell
   npm install
   npm run build
   ```

2. **Найдите папку плагинов Obsidian:**

   **Способ 1 (через настройки):**
   - В Obsidian: Настройки → Общие → Внизу найдите "Открыть папку хранилища" или "Open vault folder"
   - Перейдите в `.obsidian\plugins\`

   **Способ 2 (через проводник Windows):**
   - Откройте проводник Windows
   - В адресной строке введите: `%APPDATA%\Obsidian\plugins\`
   - Или найдите: `C:\Users\ВашеИмя\AppData\Roaming\Obsidian\plugins\`

   **Способ 3 (через папку хранилища):**
   - Найдите папку вашего хранилища Obsidian (где находятся ваши заметки)
   - Внутри должна быть скрытая папка `.obsidian`
   - Перейдите в `.obsidian\plugins\`

3. **Скопируйте плагин:**
   - Скопируйте всю папку проекта в `.obsidian\plugins\text-enhancer\`
   - Должны быть файлы: `main.js`, `manifest.json`, `styles.css`

4. **Включите плагин:**
   - Настройки → Сторонние плагины → Установленные плагины
   - Найдите "Text Enhancer" → Включить
   - Если не видите - перезапустите Obsidian

5. **Настройте API ключ:**
   - Настройки → Text Enhancer → Введите API ключ

## Где искать папку хранилища

Если не видите опцию "Хранилища" в настройках:
- Посмотрите в **Общие** (General) - там может быть кнопка "Открыть папку хранилища"
- Или найдите папку, где у вас лежат заметки `.md` - это и есть хранилище
- Внутри хранилища должна быть папка `.obsidian` (может быть скрыта - включите показ скрытых файлов)
