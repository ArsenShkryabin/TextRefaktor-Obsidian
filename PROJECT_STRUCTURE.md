# 📁 Структура проекта Text Enhancer

## Структура репозитория на GitHub

```
TextRefaktor-Obsidian/
├── src/                          # Исходный код плагина
│   ├── api/
│   │   └── AIService.ts          # Сервис для работы с AI API
│   ├── settings/
│   │   └── SettingsTab.ts        # Вкладка настроек
│   ├── ui/
│   │   └── EnhancementModal.ts   # Модальное окно предпросмотра
│   └── types.ts                  # Типы и интерфейсы
│
├── docs/                         # Документация
│   ├── ARCHITECTURE.md           # Архитектура проекта
│   ├── INSTALL_WINDOWS.md        # Установка на Windows
│   ├── БЫСТРЫЙ_СТАРТ.md          # Быстрый старт
│   ├── ТЕСТИРОВАНИЕ.md           # Полное тестирование
│   ├── ТЕСТИРОВАНИЕ_БЕЗ_API.md   # Тестирование без API
│   ├── ИДЕИ_УЛУЧШЕНИЙ.md         # Идеи для улучшения
│   └── ОБНОВЛЕНИЕ.md             # Инструкции по обновлению
│
├── main.ts                       # Главный файл плагина
├── manifest.json                 # Метаданные плагина
├── package.json                  # Зависимости npm
├── tsconfig.json                 # Конфигурация TypeScript
├── esbuild.config.mjs            # Конфигурация сборки
├── styles.css                    # Стили плагина
├── README.md                     # Основная документация
├── LICENSE                       # Лицензия MIT
├── CONTRIBUTING.md               # Руководство по вкладу
├── .gitignore                    # Игнорируемые файлы
├── .gitattributes                # Атрибуты Git
└── .editorconfig                 # Настройки редактора
```

## Что НЕ включено в репозиторий

- `scripts/` - Скрипты для разработки (остаются локально)
- `*.bat` - Батники для автоматизации (остаются локально)
- `*.mjs` - Скрипты Node.js (кроме esbuild.config.mjs)
- `node_modules/` - Зависимости npm
- `main.js` - Скомпилированный файл
- `plugin-files/` - Готовые файлы для установки
- `.obsidian/` - Настройки Obsidian

## Локальная структура (для разработки)

```
проект_расширение_обсидиан/
├── src/              # Исходный код (в репозитории)
├── docs/             # Документация (частично в репозитории)
├── scripts/           # Скрипты (НЕ в репозитории, только локально)
│   ├── copy-files.mjs
│   ├── copy-to-plugin.bat
│   └── organize-project.mjs
├── plugin-files/     # Готовые файлы (НЕ в репозитории)
├── node_modules/     # Зависимости (НЕ в репозитории)
└── [конфигурация]    # package.json, tsconfig.json и т.д.
```

## Команды для работы

```bash
# Установка зависимостей
npm install

# Сборка плагина
npm run build

# Режим разработки
npm run dev
```
