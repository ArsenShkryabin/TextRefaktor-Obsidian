===========================================
  ФАЙЛЫ ДЛЯ УСТАНОВКИ ПЛАГИНА
===========================================

Эти 3 файла нужно скопировать в папку плагина:

1. main.js - основной код плагина
2. manifest.json - метаданные плагина  
3. styles.css - стили интерфейса

===========================================
  АВТОМАТИЧЕСКОЕ ОБНОВЛЕНИЕ
===========================================

После выполнения команды:
  npm run build

Файлы автоматически копируются в эту папку!

===========================================
  ИНСТРУКЦИЯ ПО УСТАНОВКЕ
===========================================

1. Найдите папку вашего хранилища Obsidian
   (где лежат ваши заметки .md)

2. Включите показ скрытых файлов в проводнике Windows

3. Откройте папку:
   .obsidian\plugins\

4. Создайте папку text-enhancer (если её нет)

5. Скопируйте ВСЕ 3 файла из этой папки в:
   .obsidian\plugins\text-enhancer\

6. Перезапустите Obsidian

7. Включите плагин в настройках:
   Настройки → Сторонние плагины → Text Enhancer → Включить
