===========================================
  УСТАНОВКА ПЛАГИНА TEXT ENHANCER
===========================================

В этой папке находятся 3 файла для установки плагина:

✅ main.js          - основной код плагина
✅ manifest.json    - метаданные плагина  
✅ styles.css        - стили интерфейса

===========================================
  КАК УСТАНОВИТЬ
===========================================

1. Найдите папку вашего хранилища Obsidian
   (где лежат ваши заметки .md)

2. Включите показ скрытых файлов в проводнике:
   Вид → Показать → Скрытые элементы

3. Откройте папку:
   .obsidian\plugins\

4. Создайте папку text-enhancer (если её нет)

5. Скопируйте ВСЕ 3 файла из этой папки в:
   .obsidian\plugins\text-enhancer\

6. Перезапустите Obsidian

7. Включите плагин:
   Настройки → Сторонние плагины → Text Enhancer → Включить

8. Настройте API ключ:
   Настройки → Text Enhancer → Введите API ключ

===========================================
  НОВЫЕ ВОЗМОЖНОСТИ
===========================================

✨ Настройка горячих клавиш прямо в настройках плагина
✨ Кнопка "Тест API" для проверки подключения
✨ Кнопка "Тест" для проверки горячих клавиш
