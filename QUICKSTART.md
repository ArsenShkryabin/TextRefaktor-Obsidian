# Быстрый старт

## Установка

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Соберите плагин:**
   ```bash
   npm run build
   ```

3. **Скопируйте папку плагина в Obsidian:**
   - Скопируйте всю папку проекта в `.obsidian/plugins/text-enhancer/` вашего хранилища Obsidian
   - Или создайте символическую ссылку для разработки

4. **Включите плагин:**
   - Откройте Obsidian
   - Перейдите в Настройки → Сообщество плагинов → Установленные плагины
   - Найдите "Text Enhancer" и включите его

5. **Настройте API ключ:**
   - Откройте Настройки → Text Enhancer
   - Введите ваш OpenAI API ключ
   - Настройте другие параметры при необходимости

## Использование

1. Выделите текст в редакторе Obsidian
2. Нажмите `Ctrl+P` (или `Cmd+P` на Mac) для открытия палитры команд
3. Введите "Улучшить текст" и выберите нужную команду:
   - **Улучшить текст (исправить и структурировать)** - исправляет ошибки и улучшает структуру
   - **Улучшить и дополнить текст** - делает то же самое + добавляет свои мысли

4. Настройте горячие клавиши в Настройки → Горячие клавиши

## Разработка

Для разработки с автоматической пересборкой:
```bash
npm run dev
```

Это будет автоматически пересобирать плагин при изменении файлов.
